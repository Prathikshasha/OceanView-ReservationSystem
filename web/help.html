Help - Hotel Reservation System

This system allows hotel staff to manage room reservations and billing.

How to use

  • Login: Enter your username and password to access the dashboard.
  • Add Reservation: Fill the reservation form and click “Save Reservation”.
  • View Reservation: Search using the reservation number to view details.
  • Calculate Bill: Enter reservation number to generate total bill and print.
  • Logout: Click logout to end the session securely.

Common Issues

  • If login fails, check your username/password.
  • If reservation not found, verify the reservation number.
  • Check-out date must be after check-in date.
